# ARUS Storefront

Prototipe toko online yang bagian "trending"-nya digerakkan oleh data
(lihat komentar `DATA LAYER` di `src/App.jsx`).

## Coba di komputer sendiri (opsional, butuh Node.js)

```
npm install
npm run dev
```

## Deploy ke publik (tanpa coding, lihat panduan dari Claude)

1. Upload semua file di folder ini ke repository GitHub baru.
2. Hubungkan repository itu ke Vercel ("Add New Project" → pilih repo → Deploy).
3. Vercel otomatis menjalankan `npm install` dan `npm run build`, lalu
   memberi URL publik seperti `https://nama-project.vercel.app`.
